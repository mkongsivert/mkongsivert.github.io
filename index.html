Group Project Papers

Understanding social welfare loss in homogeneous vs. heterogeneous dynamic matching markets

In this project, we implement a simple model of a dynamic matching market, extending the simple greedy model to one that may alternate strategies at any time. We then analyze the results and propose extensions for more realistic settings.

MAGE-SELECT: scRNA-seq marker gene selection as a multi-Level Bayesian logistic regression problem

MAGE-SELECT is an approach to identifying "marker" genes (which distinguish cell types from each other) by grouping them using multi-level Bayesian logistic regression. We also show in this document that it is both efficient and more accurate than several alternative strategies.
